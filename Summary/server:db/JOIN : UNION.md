# UNION vs UNION ALL

둘의 차이는 UNION ALL의 경우 중복을 고려하지 않고 전부 나온다. UNION은 중복행이 있건 없건 중복검사를 수행하기 때문에 UNION ALL이 훨씬 빠르다.

| 이름         | 분류   |
| ------------ | ------ |
| 유재석       | 연예인 |
| 강호동       | 연예인 |
| 박명수       | 연예인 |
| 볼빨간사춘기 | 연예인 |

| 이름         | 분류     |
| ------------ | -------- |
| 태연         | 여자가수 |
| 유주         | 여자가수 |
| 이해리       | 여자가수 |
| 볼빨간사춘기 | 여자가수 |

select name from celebrity
**union**
select name from female_singer

| 이름         |
| ------------ |
| 유재석       |
| 강호동       |
| 박명수       |
| 볼빨간사춘기 |
| 태연         |
| 유주         |
| 이해리       |

select name from celebrity
**union all**
select name from female_singer

| 이름         |
| ------------ |
| 유재석       |
| 강호동       |
| 박명수       |
| 볼빨간사춘기 |
| 태연         |
| 유주         |
| 이해리       |
| 볼빨간사춘기 |
